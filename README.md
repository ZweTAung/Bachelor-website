# Bachelor Website – Gruppe 2 

Dette repositoriet inneholder nettsiden for bachelorprosjektet til **Gruppe 2** ved
Universitetet i Sørøst-Norge (USN).

Nettsiden brukes til å presentere:
- Prosjektbeskrivelse
- Gruppemedlemmer
- Status og ukentlige oppdateringer
- Generell prosjektinformasjon

---


Prosjektet er utviklet som en statisk nettside ved bruk av:

- **HTML5**
- **CSS3**
- **Git & GitHub** for versjonskontroll og samarbeid

Ingen rammeverk eller eksterne biblioteker er brukt.

---


Prosjektet benytter en strukturert branching-strategi:

main
│
└── development
    │
    └── feature/*
